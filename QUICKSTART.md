# Quick Start Guide

Get up and running in 5 minutes!

## 1. Install Dependencies

```bash
npm install
```

## 2. Start the App

```bash
npm start
```

The app will open at http://localhost:3000

## 3. Login

- Enter your Tick email and password
- Select a subscription (if you have multiple)

## 4. Start Tracking Time!

- Click `+ Add` on any day to create an entry
- Drag entries between days to move them
- Hold `Ctrl` while dragging to copy

## Key Features

✅ Horizontal timeline view  
✅ Drag & drop entries  
✅ Visual hours display  
✅ Add/edit/delete entries  
✅ Create repeated entries  
✅ Works offline (PWA)
