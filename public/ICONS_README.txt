PLACEHOLDER FOR ICONS
=====================

This directory should contain the following icon files for the PWA:

1. favicon.ico - Browser tab icon (16x16, 32x32, 64x64)
2. logo192.png - App icon for mobile (192x192)
3. logo512.png - App icon for mobile (512x512)

You can generate these icons from any logo or image using online tools:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/

For a quick start, you can:
1. Create a simple square image (512x512) with your app branding
2. Use the image as both logo512.png and logo192.png (resize to 192x192)
3. Convert to .ico format for favicon.ico

Recommended colors for Tick Time Tracker:
- Primary: #667eea (Purple)
- Background: #ffffff (White)

Temporary solution:
You can create simple colored squares as placeholders until you have proper icons.
The app will work without these files, but users may see broken image icons.
